Conversor de CSV para JSON.

## Introdução

Ensinando como implantar uma lib no Pypi na PUC usando poetry e twine.


### O que este projeto pode fazer

Leia um arquivo **csv** ou uma **pasta** com csvs e converta-os em **JSON**
Este projeto é um programa rodando no terminal, de preferência instale com pipx:



``
pipx install vtr-csv-json-converter

``



Para usar, basta digitar:

```
csv_converter --help

```



Isso listará todos as opções disponíveis.
